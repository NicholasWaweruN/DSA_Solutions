# PracticeSplit

This folder contains the merged practice project split into separate C# files:

- Program.cs
- StringAlgorithms.cs
- ArrayAlgorithms.cs
- ListAlgorithms.cs
- StackQueueAlgorithms.cs
- SlidingWindowAlgorithms.cs
- PrefixSumAlgorithms.cs
- LinqProblems.cs
- GlobalUsings.cs

## Suggested project setup

Create a console project and copy these files into it:

```bash
dotnet new console -n Practice
```

Then replace the generated Program.cs and add the rest of the files.

- The code was consolidated and cleaned from the uploaded practice files.
